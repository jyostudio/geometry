    静态barycentric

    • barycentric(
          value1: number,
          value2: number,
          value3: number,
          amount1: number,
          amount2: number,
      ): number

      返回给定三角形定义的某个点在一个轴上的 Cartesian 坐标，以及两个法线化质心（重心）坐标。

      参数

      • value1: number

        定义三角形的顶点 1 在一个轴上的坐标。

      • value2: number

        定义三角形的顶点 2 在一个轴上的坐标。

      • value3: number

        定义三角形的顶点 3 在一个轴上的坐标。

      • amount1: number

        法线化质心（重心）坐标 b2，等于顶点 2 的权重因子，该顶点的坐标已在 value2 中指定。

      • amount2: number

        法线化质心（重心）坐标 b3，等于顶点 3 的权重因子，该顶点的坐标已在 value3 中指定。

      返回 number

      给定三角形定义的某个点在一个轴上的 Cartesian 坐标，以及两个法线化质心（重心）坐标。
