静态barycentric

  • barycentric(value1, value2, value3, amount1, amount2): Number

  • 返回给定三角形定义的某个点在一个轴上的 Cartesian 坐标，以及两个法线化质心（重心）坐标。

    参数

    • value1: Number

      定义三角形的顶点 1 在一个轴上的坐标。

    • value2: Number

      定义三角形的顶点 2 在一个轴上的坐标。

    • value3: Number

      定义三角形的顶点 3 在一个轴上的坐标。

    • amount1: Number

      法线化质心（重心）坐标 b2，等于顶点 2 的权重因子，该顶点的坐标已在 value2 中指定。

    • amount2: Number

      法线化质心（重心）坐标 b3，等于顶点 3 的权重因子，该顶点的坐标已在 value3 中指定。

    返回 Number

    给定三角形定义的某个点在一个轴上的 Cartesian 坐标，以及两个法线化质心（重心）坐标。
